# Section 4: Programming Basics

## Overview

In this section, you'll learn programming fundamentals by building a complete Pet Management Application. Each concept is introduced and immediately applied in practical exercises.

## Development Workflow

### Typical Cycle

```
1. Read the exercise instructions
2. Write the code in VSCode
3. Save the file (Ctrl+S / Cmd+S)
4. Check the browser for results
5. If there's an error:
   - Read the error message
   - Check the console (F12 → Console)
   - Fix the error
   - Save and check again
6. When it works, move to next exercise
```
